What are some ways to implement variable fonts into a website?

To download these demos, go to the GitHub repository

Dreidel: Manually animate the variable axes in a loop That Then This: Manually animate the variable axes in a loop, but design the font so that each letter is a little different Music Box: Change the animation to look more like stop motion PowerPack: Animate each letter’s variable axes individually (with different duration and delays) Bashful: Trigger variable animation on mouse hover Mini Mochi: Repeatedly randomize every letter’s variable values and animate the transition Caffeine: Repeatedly randomize every letter’s variable values, but DON’t animate the transition
